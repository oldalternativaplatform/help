Документация в Eclipse

Документация может быть подключена к Eclipse в виде плагина. Плагин можно установить из Eclipse, указав в качестве сайта обновлений адрес http://help.alternativaplatform.com. 		Eclipse help

Documentation now can be integrated into Eclipse as a plug-in. It can be install directly from Eclipse, by setting address http://help.alternativaplatform.com as an update website.

Eclipse 3.4: Help > Software Updates > Available Software > Add Site…

Документация во Flash CS3

Для подключения документации скопируйте содержимое архива (.zip | 454 Kb) в каталог C:\Documents and Settings\All Users\Application Data\Adobe\Flash CS3\en\Configuration\HelpPanel\Help\Alternativa3D.
